# TaskFlow - Ứng dụng Quản lý Công việc (To-Do List)

## Tính năng chính

1. **Đăng ký, đăng nhập** - Xác thực người dùng để truy cập ứng dụng
2. **Thêm, sửa, xóa công việc** - CRUD operations đầy đủ
3. **Đánh dấu trạng thái công việc** - 3 trạng thái: Todo → Doing → Done
4. **Lọc công việc theo trạng thái** - All, Todo, Doing, Done
5. **Lưu trữ Local Storage** - Dữ liệu được lưu trên trình duyệt
6. **Độ ưu tiên công việc** - High, Medium, Low với màu sắc phân biệt
7. **Phân loại danh mục** - Work, Personal, Shopping, Health, Education, Other
8. **Tìm kiếm theo từ khóa** - Tìm kiếm real-time
9. **Form validation đầy đủ** - Kiểm tra dữ liệu đầu vào với thông báo lỗi
10. **Custom Select với animation** - Dropdown mượt mà, bo tròn
